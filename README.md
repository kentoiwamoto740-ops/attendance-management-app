# 勤怠管理システム

## 概要
出勤・退勤・休憩を管理し、勤務時間を自動計算できるWebアプリです。  
また、勤怠データをCSV形式でダウンロードすることができます。

## 使用技術
- Java 17
- Spring Boot
- MySQL
- Thymeleaf
- HTML / CSS
- Maven

## 機能一覧
- ログイン機能
- ユーザー登録機能
- 出勤 / 退勤機能
- 休憩開始 / 終了機能
- 勤務時間自動計算機能
- 勤怠履歴表示機能
- CSVダウンロード機能

## 工夫した点
- BCryptを用いてパスワードを暗号化し、セキュリティを意識した実装にしました。
- 勤務時間から休憩時間を差し引くロジックを実装し、実務に近い計算ができるようにしました。
- CSV出力時に文字化けが発生しないよう、UTF-8（BOM付き）で出力するよう対応しました。

## 今後の課題

- ユーザーが直感的に操作できるよう、UI/UXの改善を行っていきたいです。

## 開発環境
- IntelliJ IDEA
- Git / GitHub

## 起動方法
1. リポジトリをクローン
2. MySQLを起動し、データベースを作成
3. application.propertiesにDB接続情報を設定
4. Spring Bootアプリを起動
5. http://localhost:8080 にアクセス
